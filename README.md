NGFW (Next Generation Firewall) - межсетевой экран для глубокой фильтрации трафика, интегрированный с IDS/IPS и обладающий возможностью контролировать и блокировать трафик на уровне приложений.
Минусы NGFW - Высокая стоимость, высокая нагрузка на оборудование и сложная настройка/поддержка.
Традиционный Firewall - межсетевой экран, обеспечивающий фильтрацию только по IP/портам.
Минусы Firewall - снижение производительности системы, ложные срабатывания, блокирующие легитимный трафик, и неспособность защитить от сложных угроз нулевого дня, внутренних атак или шифрованного трафика.
В цветочном магазине можно использовать обычный Firewall так как его легче настраивать. В магазинах или офисах обычно на одном компьютере стоит одна программа (магазин приложение для учёта/сканирования товара, офис 1с) и если ограничить установку сторонних приложений (настройка/установка приложений будет происходить только администратором по логину и паролю), то тогда, ограничивать приложения не понадобится.
Remote access VPN - это технология, создающая зашифрованный туннель через интернет для подключения удаленных пользователей к корпоративной сети. Соединяет устройство пользователя с сервером.
Минусы Remote access VPN - снижение скорости интернета из-за шифрования, сложную настройку для крупных компаний, риск компрометации домашних устройств
Site-to-Site VPN - создаёт зашифрованный туннель, связывающий, локальные офисные сети между собой, а не отдельные устройства как Remote access VPN. 
Минусы Site-to-Site VPN - высокая сложность настройки, необходимость квалифицированного персонала, зависимость от качества интернет-провайдеров и высокие затраты на специализированное оборудование
В моём примере 1 магазин, а не сеть магазинов так что реализовывать Site-to-Site VPN негде. А Remote access VPN подойдёт для удалённой работы офисного персонала. Они смогут подключаться из дома к серверу защищённым групповыми политиками.
SWG (Secure Web Gateway - безопасный веб-шлюз) - это прокси-решение для сетевой безопасности, работающее между пользователями и интернетом. Оно фильтрует трафик, блокирует вредоносные сайты, фишинг и контролирует соблюдение политик компании, защищая корпоративную сеть от внешних угроз.
Минусы SWG (Secure Web Gateway - задержки трафика из-за инспекции, высокая нагрузка при расшифровке SSL/TLS, сложность настройки и обслуживания, а также риски конфиденциальности, связанные с просмотром трафика сотрудника. Они уступают NGFW в защите периметра и не контролируют все протоколы.
Сетевая DLP (Network Data Loss Prevention) — это класс программно-аппаратных средств защиты от утечек конфиденциальной информации
